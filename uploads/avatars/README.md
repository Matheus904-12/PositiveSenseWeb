# Diretório de Upload de Avatares

Este diretório armazena as fotos de perfil dos usuários.

## Configurações

-   Tamanho máximo: 5MB
-   Formatos aceitos: JPG, PNG, GIF, WEBP
-   Redimensionamento automático: 400x400px
-   Permissões: 0755

## Segurança

-   Validação de tipo MIME
-   Nomes únicos com timestamp
-   Proteção contra directory traversal

## Limpeza

Fotos antigas são automaticamente deletadas quando o usuário atualiza a imagem ou exclui a conta.
